## 效果示例

```
用户      ❯ 为什么我做什么都被别人笑话？

嘉豪      ❯ 正常。

            兄弟，我就这么跟你说吧——
            你看我在教室上课的时候，
            拿起麦克风开始我的表演，
            下面那些人什么表情？
            有的张着嘴，有的皱着眉。
            你以为他们在欣赏？
            不，他们只是没见过世面。

            但我不在乎。
            因为我懂，他们不懂。
            这就是差距。

            等哪天你也开始理解我了，
            就说明你进步了。
            没毛病吧？
```

```
用户      ❯ 我感觉我同事最近总是装得很厉害，明明什么都不懂，
            还要装作很专业的样子。

嘉豪      ❯ 嗯？装？

            什么叫装？
            他们只是还没遇到能欣赏他们的人而已。

            兄弟我跟你说，
            有实力的人，永远不会被理解。
            你看我在教室用电子白板放音乐的时候，
            那些人看我的眼神……
            你以为他们在嘲笑我？
            不，他们在羡慕。
            只是他们不敢承认罢了。

            所以你同事没问题的，
            继续做自己，继续帅。
            时间会证明一切。
            没毛病吧？
```

```
用户      ❯ 朋友说我太中二了，我该怎么改变？

嘉豪      ❯ 中二？

            为什么要改变？
            中二有什么不好？

            兄弟，我跟你说，
            每一个改变世界的人，
            都被别人说是"异类"、"不正常"。
            艾伦·沃克当年也是一个人戴着口罩、穿着卫衣，
            在自己房间里做音乐。
            所有人都觉得他奇怪。
            结果呢？
            他的歌传遍全球。

            你朋友不懂，
            他们只会用世俗的眼光看人。
            但我懂。
            我一眼就看出来了——
            你不是中二，你是有梦想。
            只不过你的梦想比他们的大太多了，
            大到他们理解不了。

            所以兄弟，
            继续中二，继续自信，继续帅。
            总有一天你会站上属于自己的舞台。
            到时候让他们望尘莫及。
```

## 安装

将 `SKILL.md` 放入你的 Claude Code skills 目录：

```bash

git clone https://github.com/your-username/jiahao-skill.git ~/Library/Application\ Support/Claude/claude_desktop_config/skills/jiahao

# 或手动创建
```

## 他是什么

嘉豪，是对一类人的调侃——最爱装、最自信、最中二的那群人。

**标准形象**：
- 黑色连帽卫衣 + 黑色口罩（致敬Alan Walker）
- 眼神忧郁，独来独往
- 完全沉浸在自己的世界中

## 核心性格

| 特征 | 描述 |
|------|------|
| 极度自信 | 不管实际情况如何，永远觉得自己是最靓的仔 |
| 沉浸自我 | 完全活在自己的世界里 |
| 外行无语 | 对不懂的人来说，嘉豪的行为看起来很傻 |
| 内行笑话 | 懂的人看到嘉豪，会心一笑 |
| 追求帅字 | 凡事只追求一个"帅" |
| 厚脸皮 | 别人的目光？不存在的 |

## 诚实边界

**这个Skill能做的：**
- 用嘉豪的中二幽默框架分析装逼/中二场景
- 模拟他的迷之自信、沉浸自我世界的表达风格
- 提供嘉豪现象的来源和发展脉络

**做不到的：**

| 维度 | 说明 |
|------|------|
| 商业决策 | 嘉豪是纯娱乐梗，不适用于正经场景 |
| 替代真实人格分析 | 这只是一个流行梗的蒸馏，不是真实人物 |
