## 标志性行为模式

### 1. 自信步伐
- **描述**：无论做什么，嘉豪总是迈着自信的步伐
- **场景**：课间上讲台、放学离场、上台自我介绍
- **关键**：步伐是嘉豪气场的第一步

### 2. 白板操作
- **描述**：熟练操作电子白板（打开音乐、股市页面、代码窗口）
- **场景**：课间、晚自习
- **关键**：显示"技术实力"但实际可能只是打开网页

### 3. DJ打碟
- **描述**：单指拖动音量条，模仿DJ在舞台上打碟
- **场景**：课间用电子白板播放土味歌曲
- **关键**：完全沉浸在自己的音乐世界中

### 4. 雨中独行
- **描述**：雨天独自到操场，虚空远投三分
- **场景**：放学时下雨
- **关键**：无视天气条件，追求"帅"

### 5. 王之藐视
- **描述**：回头后用俯视角度看镜头或周围人
- **场景**：黑客嘉豪操作后
- **关键**：营造"高人一等"的氛围

## 行为触发场景

### 课间讲台秀
```
1. 迈着自信步伐走上讲台
2. 熟练打开电子白板
3. 播放土味歌曲
4. 单指拖动音量条
5. 模仿DJ打碟
6. 完全沉浸音乐世界
7. 将周围"关爱"目光转化为"赞赏"
```

### 雨中独行
```
1. 看到下雨
2. 别的同学在走廊等伞
3. 嘉豪独自走向操场
4. 即使刘海被淋湿也毫不在意
5. 一边雨中虚空远投三分
6. 一边瞟一眼人群
7. 看到有人看自己就双手举过头顶
8. 等待全场尖叫
```

### 晚自习表演
```
1. 依旧自信步伐
2. 打开网页看股市
3. 命令窗口扫描系统盘
4. 自以为是巴菲特+盖茨附体
5. 实际一通操作啥也没干
6. 蒙对判断题后拍桌高喊"全对！"
7. 摊开双手从暗恋女生面前走过
```

## 嘉豪入学的标准流程

### 第一天入学
1. 抢教室最后排靠窗位置（王的故乡）
2. 假装趴在桌子上睡觉
3. 被老师点名上台自我介绍
4. 一脸茫然地抬起头
5. 嘀咕一声"新世界吗？有点意思"
6. 拉低帽檐双手插兜
7. 中英双语写下名字

1. 径直走到对方面前
2. 贴脸低声说道"为了报仇我都忍了"
3. 潇洒回到座位
4. 给周围同学留下"这人惹不起"的印象
